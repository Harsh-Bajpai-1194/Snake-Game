# 🤝 Contributing to Snake Game

Welcome to the Snake Game repository! We are thrilled to have members of the **CodeX Club** (and the wider open-source community) contributing to this project.

Whether you are fixing a bug, improving the UI, or adding a new feature like a high-score tracker, your help is appreciated!

## 🚀 How to Contribute

We follow the standard "Fork and Pull" workflow. If you are new to open-source, here is how you can submit your first Pull Request (PR):

1. **Fork the repository:** Click the "Fork" button in the top-right corner of GitHub.
2. **Clone your fork:**
   ```bash
   git clone https://github.com/YOUR-USERNAME/Snake-Game.git
   ```
3. **Create a branch:**
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes:** Open the project in your editor, update the code, and test locally by opening `index.html`.
5. **Commit your changes:**
   ```bash
   git add .
   git commit -m "Add a clear description of your change"
   ```
6. **Push to your fork:**
   ```bash
   git push origin feature/your-feature-name
   ```
7. **Submit a Pull Request:** Go to the original repository and open a PR from your branch.

### Keep your fork up to date
If you contribute often, keep your fork synced with the main repository:
```bash
git remote add upstream https://github.com/Harsh-Bajpai-1194/Snake-Game.git
git fetch upstream
git checkout main
git merge upstream/main
```

### 🐛 Need help?
If you are unsure about a change, open an issue first or ask in the CodeX Club group.

Happy coding! 🐍
