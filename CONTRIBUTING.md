# 🤝 Contributing to Snake Game

Welcome to the Snake Game repository! We are thrilled to have members of the **CodeX Club** (and the wider open-source community) contributing to this project. 

Whether you are fixing a bug, improving the UI, or adding a new feature like a high-score tracker, your help is appreciated!

## 🚀 How to Contribute

We follow the standard "Fork and Pull" workflow. If you are new to open-source, here is how you can submit your first Pull Request (PR):

1. **Fork the Repository:** Click the "Fork" button at the top right of this page to create a copy of this repository under your own GitHub account.
2. **Clone your Fork:** ```bash
   git clone [https://github.com/YOUR-USERNAME/Snake.git](https://github.com/YOUR-USERNAME/Snake.git)

```

3. **Create a Branch:** Create a new branch for your feature or bug fix.
```bash
git checkout -b feature/your-feature-name

```


4. **Make your Changes:** Open the code in your editor, write your code, and test it locally by opening `index.html`.
5. **Commit your Changes:**
```bash
git add .
git commit -m "Added a cool new feature"

```


6. **Push to your Fork:**
```bash
git push origin feature/your-feature-name

```


7. **Submit a Pull Request:** Go back to the original repository on GitHub, and you will see a prompt to compare and submit a Pull Request.

## 🐛 Finding Something to Work On

Check out the **Issues** tab. Look for issues tagged with `Good First Issue` if you are just getting started!

Happy Coding! 🐍

```

## 🎮 How to Play
- Use the **Arrow Keys** (⬅️ ⬆️ ➡️ ⬇️) to move the snake.
- Eat the red square (🍎) to grow longer.
- When the yellow gate appears, go through it to clear the level.
- Avoid colliding with the walls or yourself. 
- The game ends when you crash!
